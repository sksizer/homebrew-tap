# Sksizer Tap

## How do I install these formulae?

`brew install sksizer/tap/<formula>`

Or `brew tap sksizer/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "sksizer/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
